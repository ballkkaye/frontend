# ⚾ 파이널 프로젝트 3 : KBO 팬들을 위한 올인원 커뮤니티 플랫폼

- Flutter로 개발한 야구 커뮤니티 기반 앱입니다.
- 경기 예측, 직관 동행, 커뮤니티 등 다양한 기능을 통합하여 팬들이 함께 즐길 수 있는 모바일 플랫폼을 제작했습니다.

<br>

## 🔨 주요 기능

| 카테고리         | 구현 기능                               |
|--------------|-------------------------------------|
| **홈**        | 오늘의 경기 정보 / 예측 슬라이더 / 경기별 승리 예측     |
| **승부예측**     | 경기별 승부예측 등록 및 결과 보기 / 예측 저장 및 서버 통신 |
| **직관 동행 매칭** | 동행 글 등록 / 목록 조회 / 상세보기 / 신청 UI      |
| **커뮤니티**     | 게시글 작성 / 수정 / 삭제 / 상세보기 / 댓글 / 검색   |
| **이력서 추천**   | 구직자 이력서 기반 추천 공고 보기                 |
| **채용공고**     | 직무, 경력, 기술스택으로 조건 검색 / 공고 리스트 뷰     |
| **마이페이지**    | 내 정보 수정 / 내 활동 내역 확인 (게시글, 댓글, 지원)  |
| **로그인 인증**   | JWT 기반 자동 로그인 및 라우팅 처리              |
| **채팅 UI**    | 직관 동행 시 채팅방 레이아웃 설계                 |
